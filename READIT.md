# Sentence Analyzer

A simple Python program that analyzes a sentence entered by the user and displays useful statistics.

## Description
This project takes a sentence as input and calculates:
- Total characters
- Total words
- Number of vowels
- Number of consonants

## Features
- Beginner-friendly
- Uses string methods
- Clear and readable output
- No external libraries required

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Open terminal in the project folder
3. Run:
   python sentence_analyzer.py
4. Enter a sentence when prompted
